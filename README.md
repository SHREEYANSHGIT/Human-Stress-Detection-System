# 🧠 Human Stress Detection Using Artificial Neural Networks

This project predicts human stress levels based on health and lifestyle data using a deep learning (ANN) model built with TensorFlow and Keras.

## 🚀 Project Overview
- Preprocessed data using Pandas and Scikit-learn
- Encoded categorical and binary features
- Scaled numerical features using StandardScaler
- Designed an ANN with ReLU hidden layers and Sigmoid output
- Achieved ~85% prediction accuracy on test data

## 📊 Tech Stack
Python | TensorFlow | Scikit-Learn | Pandas | Matplotlib | Seaborn

## 📈 Model Performance

Accuracy: 85%

Model: ANN (2 hidden layers, ReLU, dropout, sigmoid output)
